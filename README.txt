Este repositorio contiene el código fuente del sitio web de la Escuela de Fútbol Alianza Dinamarca, una escuela de fútbol cristiana para niños y jóvenes en Temuco, Chile.

## Descripción

El sitio web de la Escuela de Fútbol Alianza Dinamarca es una aplicación de una sola página construida con HTML, CSS y JavaScript. Proporciona información sobre los programas de la escuela, entrenadores, horarios de entrenamiento y proceso de inscripción.

## Características

- Diseño responsivo
- Información sobre diferentes programas de fútbol para varios grupos de edad
- Perfiles de los entrenadores
- Detalles de horarios y ubicación de los entrenamientos
- Formulario de inscripción para nuevos estudiantes
- Sección de preguntas frecuentes (FAQ)
- Formulario de contacto

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3.0
- Font Awesome 6.0.0-beta3
- Google Fonts (Roboto, Poppins)
- Varias bibliotecas JS (AOS, GLightbox, Swiper, etc.)

## Configuración

Para ejecutar este proyecto localmente:

1. Clona el repositorio
2. Abre `index.html` en tu navegador web

## Estructura

- `index.html`: Contenido principal de la página
- `template_contenido.html`: Para la creación de contenido extra mencionados como sugerencia en el navbar
- `assets/`: Directorio que contiene archivos CSS, JS, imágenes y archivos de proveedores
  - `css/`: Archivos CSS personalizados
  - `js/`: Archivos JavaScript personalizados
  - `img/`: Imágenes utilizadas en el sitio web
  - `vendor/`: Bibliotecas y frameworks de terceros
